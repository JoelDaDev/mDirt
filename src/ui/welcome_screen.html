Welcome to mDirt

(making Datapacks is really tedious... But not anymore!)

mDirt is a powerful and user-friendly datapack editor for Minecraft, designed to make creating, editing, and managing datapacks easier than ever.

Getting Started

  • Create a new datapack project
  • Use built-in templates and generators
  • Generate your pack

Features

  • Drag-and-drop file management
  • Add custom Blocks, Items, Recipes and more
